# Estrutura Condicional em Java 📞

---

Este repositório contém um exercício prático focado no uso de estruturas condicionais (`if`) para a resolução de problemas de lógica de negócio em Java.

### 🚀 Tecnologias Utilizadas

---

* **Linguagem:** Java
* **IDE:** IntelliJ IDEA
* **JDK:** 21

### 📌 O que eu pratiquei:

---

* **Estrutura Condicional:** Implementação de lógica com `if` para cálculo de excedentes.
* **Manipulação de Variáveis:** Uso de tipos `int` e `double` para cálculos financeiros.
* **Saída Formatada:** Uso do `printf` para exibição de valores monetários com duas casas decimais.
* **Internacionalização:** Uso da classe `Locale` para garantir a formatação correta de números.

---

*Estudos focados em lógica de programação e fundamentos da linguagem Java.*
